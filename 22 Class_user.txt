Напишите класс под названием User, который используется для расчета суммы, которую пользователь будет продвигать по системе ранжирования, аналогичной той, которую использует Codewars.

Бизнес-правила:
Пользователь начинает с ранга -8 и может прогрессировать до 8.
Нет 0 (нулевого) ранга. Следующий ранг после -1 равен 1.
Пользователи будут выполнять действия. Эти виды деятельности также имеют ранги.
Каждый раз, когда пользователь выполняет ранжированное действие, прогресс ранжирования пользователей обновляется на основе ранга действия
Прогресс, полученный в результате выполненного действия, зависит от того, какой текущий ранг пользователя сравнивается с рангом действия
Прогресс в ранге пользователя начинается с нуля, каждый раз, когда прогресс достигает 100, ранг пользователя повышается до следующего уровня
Любой оставшийся прогресс, полученный в предыдущем ранге, будет применен к прогрессу следующего ранга (мы не выбрасываем прогресс). Исключение составляет случай, если не осталось другого ранга для продвижения (как только вы достигнете 8-го ранга, больше не будет прогресса).
Пользователь не может продвинуться выше 8 ранга.
Единственным приемлемым диапазоном значений ранга является -8,-7,-6,-5,-4,-3,-2,-1,1,2,3,4,5,6,7,8. Любое другое значение должно вызвать ошибку.
Прогресс оценивается примерно так:

Выполнение действия, которое оценивается так же, как и у пользователя, будет стоить 3 балла
Выполнение действия, которое занимает место на один рейтинг ниже, чем у пользователя, будет стоить 1 балл
Любые выполненные действия, рейтинг которых на 2 уровня или более ниже рейтинга пользователя, будут проигнорированы
Выполнение действия с рейтингом выше, чем у текущего пользователя, ускорит продвижение по рангу. Чем больше разница между рейтингами, тем больше будет увеличиваться прогресс. Формула равна 10 * d * d, где d равно разнице в рейтинге между активностью и пользователем.
Примеры логики:
Если пользователь с рейтингом -8 завершит действие с рейтингом -7, он получит 10 прогресса
Если пользователь с рейтингом -8 завершит действие с рейтингом -6, он получит 40 прогресса
Если пользователь с рейтингом -8 завершит действие с рейтингом -5, он получит 90 прогресса
Если пользователь с рейтингом -8 завершит действие с рейтингом -4, он получит 160 достижений, в результате чего пользователь будет повышен до ранга -7 и заработает 60 достижений для своего следующего ранга
Если пользователь с рейтингом -1 завершит действие с рейтингом 1, он получит 10 прогресса (помните, нулевой ранг игнорируется).
Примеры использования кода:
пользователь = Пользователь()
пользователь.ранг # => -8
пользователь.прогресс # => 0
пользователь.inc_progress(-7)
user.progress # => 10
user.inc_progress(-5) # добавит 90
пользователей прогресса.прогресс # => 0 # прогресс теперь равен нулю.
ранг # => -7 # ранг был повышен до -7
Примечание: Codewars больше не использует этот алгоритм для своей собственной системы ранжирования. В нем используется чисто математическое решение, которое дает последовательные результаты независимо от того, в каком порядке выполняется набор ранжированных действий.
